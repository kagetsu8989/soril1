## Primary Key ба Foreign Key-ийн ялгаа юу вэ? Яагаад хүснэгтүүдийг FK-ээр холбодог вэ?
- Primary Key нь хүснэгтийн мөр бүрийг давтагдашгүй гаргана.
- NULL байж болохгүй.

- Foreign Key нь өөр хүснэгтийн Primary key-г зааж холбоо үүсгэнэ.

- Яагаад хүснэгтүүдийг FK-ээр холбодог вэ?
1 өгөгдлийн холбоо үүсгэнэ
2 алдаатай өгөгдлөө хамгаална
3 өгөгдлийн бүрэн бүтэн байдал
4 JOIN хийхэд амархан
## COUNT функц болон GROUP BY-г хамтад нь яагаад ашигладаг вэ? Жишээгээр тайлбарла.
 
 SELECT book_id, COUNT(*) AS borrow_count
 from borrow_recortds
 GROUP BY book_id;

 үр дүн:

 book_id | borrow_count
 ----------------------
 1       |  3
 ----------------------
 2       |  5

 Group by байхгүй бол
 SELECT COUNT(*) FROM borrow_records;

## Өгөгдлийн сангийн нөөц (backup) авах нь яагаад чухал вэ? Ямар тохиолдолд хэрэг болдог вэ?
1. өгөгдөл алдагдахаас хамгаална
2. систем гэмтэлд бэлэн байна
3. аюулгүй байдлын хамгаалалт
4. update хийхийн өмнө backup авна.
